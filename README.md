# Benflux DevTools 🧰

Benflux DevTools is a community-driven platform where developers build useful tools together while competing, learning, and earning rewards.

Powered by **Benflux DevArena**, this project introduces a new way to contribute to open source through structured weekly challenges.

---

## 🚀 Concept

Instead of random contributions, developers participate in **weekly challenges**, where each week focuses on building or improving a specific developer tool.

Examples of tools:

- JSON Formatter
- JWT Decoder
- Password Generator
- Markdown Preview
- Regex Tester
- API Tester

Each tool is built collaboratively by the community.

---

## 🏆 How it works

1. A new challenge is released every week  
2. Developers contribute via Pull Requests  
3. Submissions are reviewed and scored  
4. The best contributors are selected  
5. Rewards are distributed in **Benflux Coin (BFC)**  

---

## 💰 Rewards System

- Weekly reward pool: **100 BFC**
- Top contributors earn based on performance
- Only high-quality contributions are accepted

---

## 📊 Evaluation Criteria

Each Pull Request is scored based on:

- Code quality
- Performance
- User experience
- Clean architecture
- Documentation
